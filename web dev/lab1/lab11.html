Question 1

Design a webpage that uses different heading tags
Note: Tags like <p> cannot be written directly to display as text. The browser will recognize them as actual tags, so we should use escape characters like &lt;p&gt;.

This is a sample paragraph where we are demonstrating the paragraph tag <p>.
Lets understand the difference between strong & b tags and em & i tags.

  • strong tag — emphasizes text semantically
  • b tag — makes text bold visually
  • em tag — emphasizes text semantically
  • i tag — italicizes text visually

Character Escape Code Description
< &lt; Represents the less-than symbol
> &gt; Represents the greater-than symbol
& &amp; Represents the ampersand symbol
