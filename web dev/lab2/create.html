1. STATIC Positioning

This is the default positioning. Elements follow the normal document flow.

Box 1 - Static
Box 2 - Static (top, left properties are ignored)
Box 3 - Static
position: static; /* Default - normal flow */

2. RELATIVE Positioning

Elements are positioned relative to their normal position. Other content is not affected.

Box 1 - Relative (Normal Position)
Box 2 - Relative (Offset by top: 20px, left: 50px)
position: relative; top: 20px; left: 50px;

3. ABSOLUTE Positioning

Elements are removed from normal flow and positioned relative to nearest positioned ancestor.

Top Left
Top Right
Bottom Left
Bottom Right
position: absolute; top: 20px; left: 20px; /* Or right/bottom */

4. FIXED Positioning

Elements are fixed relative to the viewport. They stay in the same position even when scrolling.

5. DISPLAY: BLOCK

Block elements take up the full width available and always start on a new line.

Block Box 1 (full width)
Block Box 2 (full width)
Block Box 3 (full width)
display: block; /* Takes full width, new line */

6. DISPLAY: INLINE

Inline elements flow within text, only take up necessary width, and ignore width/height properties.

Inline 1 Inline 2 Inline 3 Inline 4
display: inline; /* Flows with text, ignores width/height */

7. DISPLAY: INLINE-BLOCK

Combines inline and block properties. Elements flow like inline but respect width/height and margins.

Box 1
Box 2
Box 3
display: inline-block; /* Inline flow + respects width/height */
