Learn positioning (static, relative, absolute, fixed) and display (block, inline, inline-block)

1. STATIC (Default)

‚úì Elements flow naturally, one after another
‚úì top, left, bottom, right properties are IGNORED
Static Box
position: static;

2. RELATIVE

‚úì Element stays in normal flow, but can be offset
‚úì top/left/right/bottom move it from its original position
‚úì Other elements don't move to fill the gap
Normal Position
Moved Down 15px, Right 40px
position: relative;
top: 15px; left: 40px;

3. ABSOLUTE

‚úì Element is removed from normal flow
‚úì Positioned relative to nearest positioned parent
‚úì Can place it anywhere using top, left, right, bottom
Top Left
Top Right
Bottom Left
Bottom Right
position: absolute; top: 10px; left: 10px;

4. FIXED

‚úì Element stays fixed to the VIEWPORT (window)
‚úì Doesn't scroll with the page
‚úì Great for floating buttons, navbars
üìå Look at bottom-right corner ‚Üí there's a fixed button that stays visible when scrolling!
position: fixed; bottom: 20px; right: 20px;

5. DISPLAY: BLOCK

‚úì Takes up FULL WIDTH available
‚úì Starts on a NEW LINE
‚úì Examples: <div>, <p>, <h1>
Block 1 - Full Width
Block 2 - Full Width
display: block;

6. DISPLAY: INLINE

‚úì Flows with text, doesn't start new line
‚úì width/height properties DON'T WORK
‚úì Examples: <span>, <a>, <strong>
Inline 1 Inline 2 Inline 3
display: inline;

7. DISPLAY: INLINE-BLOCK

‚úì Flows like inline (no new line)
‚úì But respects width, height, margins
‚úì Perfect for buttons and cards
Box 1
Box 2
Box 3
display: inline-block; width: 120px;

Quick Comparison

Property Flow Respects Width/Height When to Use
Static Normal Yes Default, no special positioning
Relative Normal + offset Yes Small adjustments to position
Absolute Removed Yes Precise positioning inside container
Fixed Removed (to viewport) Yes Floating buttons, sticky navbars
Block New line, full width Yes Containers, sections
Inline With text, no new line No Text-level elements, links
Inline-Block With text, no new line Yes Buttons, small cards, badges
